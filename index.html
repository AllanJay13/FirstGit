Configuring Git

  • go to vs code and click terminal --arrow new terminal
  • check if get is already installed by typing git --version in the command
  • Configure git in your local computer
    • set Username
      git congig --global user.name "username"
    • set email address
      git config --global user.email "email"
  • to initialize the project directory type git init

Git Add and Commit

  • to see the status of the file in your directory type git status
  • to track your file use the command git add
    syntax git add. for group files
    syntax git add filename for individual files
  • by running the git status again you can see the staged file is green ready to commit
  • commit the file that you have staged in your local repository by typing
    syntax git commit -m
